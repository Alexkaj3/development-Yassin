Q / What is Wazuh system?

Wazuh is an open source, end-to-end security system for threat detection, response, and analysis. You can use Wazuh with Twilio to receive security alerts via text messages, including WhatsApp messages, by configuring message sending through the appropriate actions in Wazuh.
