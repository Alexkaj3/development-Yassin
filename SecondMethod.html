A faster way to send warnings via a WhatsApp bot, you can use event management services like Zapier or Integromat. These services allow you to link different applications together without having to write your own code.

Here's how to do it using Zapier as an example:

Create an account in Zapier if you don't already have one Click “Make a Zap” to start creating a new integration Select "Wazuh" as the source application and follow the steps to select the event you wish to use (such as receiving a new alert) Select “WhatsApp” as the destination app and follow the steps to configure the WhatsApp message you want to send when the event occurs Activate your Zap and test it to make sure it is working properly. Using this method, you can set up the integration quickly and without having to write code.
